**Description:** For this assignment, we were tasked with creating a palm drone checker. This tool allows users to input words into a text field and press enter to determine whether or not the word is a palm drum. Additionally, we were required to develop a login system for a web page. Users will enter a username and password to access a different page. Below, you will see the username and password fields that need to be filled out. Once the correct credentials are entered, the user will be directed to the next page, where they will receive a message confirming a successful login.
